# TVMon-Project

## Requirements 

### python librairys
```
pip install [libName]
```
you have to install:
* pathlib
* pandas
* matplotlib
* psutil

### linux application
``` 
sudo apt-get install [applicationName]
```
you have to install:
* vlc
* tshark
* wmctrl
